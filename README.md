## 安装教程
1. 前置环境：部署 [TRSS-Yunzai](https://gitee.com/TimeRainStarSky/Yunzai)
2. 放入 `Secluded` 目录完成插件安装
3. 执行命令安装项目依赖
4. 重启云崽，自动生成配置文件
    [>>>点击查看适配器配置示例<<<](./help/sec适配器配置.png)
5. 进入sec官方群聊，下载适配系统版本的sec程序并安装
6. 启动sec程序，打开后台网页，点击左上角菜单，进入`软件设置`页面
    [>>>页面示例<<<](./help/sec配置.png)
7. 全部配置完成后重启云崽，提示连接成功即可
8. 机器人登录指令：`#sec设置1145141919`
