## 安装教程
1. 前置环境：部署 [TRSS-Yunzai](https://github.com/TimeRainStarSky/Yunzai)
2. 执行命令安装项目依赖
3. 重启云崽，自动生成配置文件
    [>>>点击查看适配器配置示例<<<](./help/sec适配器配置.png)
4. 启动sec程序，打开后台网页，点击左上角菜单，进入`软件设置`页面
    [>>>页面示例<<<](./help/sec配置.png)
5. 全部配置完成后重启云崽，提示连接成功即可
6. 机器人登录指令：`#sec设置1145141919`
